# From fresh to dry: Moth body weight changes through preservation

This repository contains the datasets used in the study **“From fresh to dry: Moth body weight changes through preservation”**, published in *Nota Lepidopterologica*. :contentReference[oaicite:0]{index=0}

The data include body weight measurements from **laboratory-reared, field-collected, and museum-preserved moth specimens**, representing common workflows in ecological and curatorial research. These datasets were used to quantify how body weight changes across **developmental stages and preservation conditions**, including live, fresh, dry, frozen, and rehydrated specimens.

Together, the datasets provide the basis for estimating **correction factors that allow fresh body weight to be approximated from preserved specimens**, improving the use of museum collections in trait-based ecological and evolutionary studies of Lepidoptera.
